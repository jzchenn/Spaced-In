## Setup Instructions

### Backend (FastAPI)
1. `cd backend`
2. `python3 -m venv venv`
3. `source venv/bin/activate` 
4. `pip install -r requirements.txt`
5. `uvicorn main:app --reload`

### Frontend (React + Vite)
1. `cd frontend`
2. `npm install`
3. `npm run dev`
